# تفعيل المزامنة والعمل دون إنترنت

1. افتح **Supabase > SQL Editor**.
2. افتح الملف `docs/ansar-offline-sync-upgrade.sql` من المشروع وانسخ محتواه كاملاً.
3. نفّذ الملف مرة واحدة. يمكن إعادة تنفيذه بأمان عند الحاجة.
4. يجب أن تظهر النتيجة:

   `ansar offline sync upgrade installed successfully`

لا تستخدم `ansar-runtime-repair.sql` لهذه الخطوة. الترحيل الجديد مستقل عنه ولا يحذف أو يعيد تسمية أي جدول أو سجل قائم.

بعد تثبيت APK الجديد وتسجيل الدخول مرة واحدة عبر الإنترنت، يحتفظ التطبيق بالجلسة والبيانات المحلية، ويعرض حالة العمليات المعلقة من قائمة الحساب عبر **المزامنة دون إنترنت**.
